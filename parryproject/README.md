Project Title : Arrow Parry Game

How To Run: Open the parryproject file and double click index.html. this is will run the program in your prefered browser.

Research Problem : a lot of simple browser games lack engaging reaction-based mechanics. This is an interactive simple game that relies on precise timing and reaction speed.

Motivation : I wanted to create a simple game with a parry mechanic that is seen in popular games like Sekio, lies of P, and Exp33. The goal was to use the skills I've aquired in this course to make something fun and interactive.

Control Flow : You start the game with 3 lives. Arrows come from the right side of the screen towards the player. The goal is press Space to parry the incoming arrows at the right time. Mistimed parries will result in the arrows hitting you, and you losing 1 life. The game ends when you reach 0 lives.

Implementation Strategy : The HTML sets up a canvas and a user interface. The CSS adds color and size to the canvas and text and styles them. The JavaScript handles the game loop, the speed and spawnrate of the arrows, the collisions, and it uses the png and mp3 files to add visuals to the player and arrows.
